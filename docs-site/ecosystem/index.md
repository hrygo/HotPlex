### Official SDKs

Build your own HotPlex integrations in your language of choice. Our SDKs follow the same core DMP protocol but focus on idiomatic patterns for each language.

::: code-group
```go [Go SDK]
// The flagship SDK for high-performance integrations.
import "github.com/hrygo/hotplex-go"

engine := hotplex.NewEngine(...)
```

```python [Python SDK]
# The favorite for data scientists and AI researchers.
from hotplex import Engine

engine = Engine(api_key="...")
```

```typescript [TypeScript SDK]
// Perfect for modern web portals and serverless triggers.
import { HotPlex } from '@hrygo/hotplex-js';

const client = new HotPlex({...});
```
:::
