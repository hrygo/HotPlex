# Go 1.25 语言特性

## 自动生效（无需改代码）
- **Swiss Table Maps**：`make(map[K]V)` 使用新哈希表
- **Container-Aware GOMAXPROCS**：自动适配容器 CPU 限制

## 推荐使用
- **`log/slog`**：标准库结构化日志，统一使用，替代第三方日志库
- **Generic Interfaces**：Worker 接口定义可利用类型参数增强抽象（如 `Worker[T Event]`）
- **`weak.Value`**：适合 session metadata LRU 缓存（避免 goroutine 泄漏）
- **`slices.Clone`**：slice 边界复制的标准方式
- **`unique.Make`**：字符串驻留，减少重复分配

## 构建优化
```bash
go build -pgo=auto ./cmd/gateway  # Profile-Guided Optimization
```

## Flight Recorder（生产诊断）
```bash
go tool trace trace.out
```
