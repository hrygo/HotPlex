# 安全政策

## 支持的版本

我们建议始终使用 HotPlex 的最新版本。

| 版本     | 是否支持           |
| -------- | ------------------ |
| v0.9.x   | :white_check_mark: |
| v0.8.x   | :white_check_mark: |
| < v0.8.0 | :x:                |

## 报告漏洞

如果您在 HotPlex 中发现了安全漏洞，请不要提交公开的 Issue。请私下报告。

您可以通过以下方式报告漏洞：
- 在 GitHub 上开启私有安全咨询 (Private Security Advisory)（如果可用）。
- 直接通过电子邮件联系维护者（待配置）。

我们将在 48 小时内确认您的报告，并提供修复时间表。在公开任何信息之前，请给我们合理的时间来调调查和解决问题。
